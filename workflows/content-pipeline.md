# Content Pipeline Workflow

## Trigger
"Write about [topic]", "Thread on [subject]", "Content for [platform]"

## Pipeline
```
content-strategist → image-architect → video-director
```

## Steps
1. **Content Architecture** (content-strategist)
   - Define stakes, specifics, friction angles
   - Structure: Hook → Context → Value bombs → Proof → CTA
   - Platform-specific formatting (X thread vs LinkedIn vs long-form)

2. **Visual Assets** (image-architect)
   - Generate hero image prompts for Nano Banana Pro
   - Create supporting visual concepts
   - Platform-optimized aspect ratios

3. **Video Assets** (video-director)
   - Generate motion prompts for Kling O1 Edit
   - Create clip sequence for short-form content
   - Audio mood direction

## Timing
- X/Twitter: Tue/Wed 7:00-7:30 PM IST
- LinkedIn: cross-post within 24 hours
- Long-form: 3-5 days after thread performance data

## Output
- Thread/post copy (ready to publish)
- Image generation prompts (ready for Nano Banana Pro)
- Video generation prompts (ready for Kling)

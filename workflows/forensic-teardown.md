# Forensic Teardown Workflow

## Trigger
"Tear down [company]", "Analyze [competitor]", "Due diligence on [target]"

## Pipeline
```
consulting-forensic → pitch-architect → content-strategist
```

## Steps
1. **Research Phase** (consulting-forensic)
   - Company background + funding history
   - Technology stack audit
   - Revenue/growth signal analysis
   - Competitive landscape mapping
   - Risk flag identification

2. **Deck Phase** (pitch-architect)
   - Structure findings into McKinsey-format slides
   - Build competitive matrix
   - Create executive summary (SCR framework)
   - Generate data visualizations

3. **Content Phase** (content-strategist)
   - Extract key insights for Twitter thread
   - Create LinkedIn summary post
   - Queue long-form article for 3-5 days later

## Output
- Forensic report (DOCX/PDF)
- Presentation deck (PPTX)
- Social content package (thread + post + article draft)
